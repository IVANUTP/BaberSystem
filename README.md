# 💈 BarberSystem

Sistema web de gestión para barbería desarrollado con Laravel.
Permite administrar citas, usuarios, servicios y blog en una sola plataforma.

---

## 🚀 Funcionalidades

* 📅 Gestión de citas (crear, editar, cancelar)
* 👤 Roles de usuario (admin, barbero, cliente)
* ✂️ Administración de servicios
* 📰 Blog integrado
* 🔔 Sistema de notificaciones
* 🖼️ Galería de imágenes
* 🔐 Autenticación de usuarios

---

## 🛠️ Tecnologías utilizadas

* **Backend:** Laravel
* **Frontend:** Blade + Bootstrap
* **Base de datos:** MySQL
* **API:** REST con Laravel
* **Otros:** Cloudinary (imágenes), Axios

## ⚙️ Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/IVANUTP/BaberSystem.git
```

2. Entrar al proyecto:

```bash
cd barberia-citas
```

3. Instalar dependencias:

```bash
composer install
npm install
```

4. Configurar entorno:

```bash
cp .env.example .env
php artisan key:generate
```

5. Configurar base de datos en `.env`

6. Ejecutar migraciones:

```bash
php artisan migrate
```

7. Levantar servidor:

```bash
php artisan serve
```
